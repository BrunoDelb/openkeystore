JSON - Encoder, Decoder and Signature Package

This package contains classes for parsing and generating messages in JSON. To make the system slightly more on par with XML Schemas the JSON library also supports:
  • Registration of JSON decoders (=supporting a specific message) and automatic instantiation during parsing
  • Strict property sequences including detection of missing reads
  • Strong validation of input

In addition, there is a set of classes supporting a [very] scaled-down JSON counterpart to enveloped XML signatures.

The signature scheme signs the JSON object identified by the Reference using the data in blue, down to and including the SignatureInfo object.
