JSON - Encoder, Decoder and Signature Package

This package contains classes for parsing and generating messages in JSON. To make the system slightly more on par with XML Schemas the JSON library also supports:
  • Registration of classes extending {@link org.webpki.json.JSONDecoder} (=supporting a specific message) and automatic instantiation during parsing. See {@link org.webpki.json.JSONDecoderCache}
  • Detection of missing reads
  • Strong validation of input

In addition, there is a set of classes supporting a [very] scaled-down JSON counterpart to enveloped XML signatures: JSON Cleartext Signature.
See {@link org.webpki.json.JSONObjectWriter#setSignature(JSONSigner)} and {@link org.webpki.json.JSONObjectReader#getSignature()} for programming information.
